<div class="main-page-block-heading">Apie Vikipediją</div>
<div class="main-page-block-contents"><b><a href="/wiki/Vikipedija" title="Vikipedija">Vikipedija</a></b> yra universali, daugiakalbė interneto enciklopedija, kaip bendruomeninis projektas, pagal <i><a href="/wiki/Wiki" class="mw-redirect" title="Wiki">viki</a></i> technologiją ir <a href="/wiki/Vikipedija:Penki_stulpai" title="Vikipedija:Penki stulpai">pamatinius principus</a> kuriama daugybės <a href="/wiki/Vikipedijos_bendruomen%C4%97" title="Vikipedijos bendruomenė">savanorių</a> bei išlaikoma iš paaukotų lėšų.
<p>Vikipedijos tikslas – pateikti <a href="/wiki/Vikipedija:Autori%C5%B3_teis%C4%97s" title="Vikipedija:Autorių teisės">laisvą</a>, <a href="/wiki/Vikipedija:Neutralus_po%C5%BEi%C5%ABris" title="Vikipedija:Neutralus požiūris">nešališką</a> ir <a href="/wiki/Vikipedija:Patikrinamumas" title="Vikipedija:Patikrinamumas">patikrinamą</a> turinį, kurį be jokių apribojimų gimtąja kalba galėtų skaityti visi žmonės. Rašyti, pildyti, tobulinti straipsnius taip pat gali visi, jei laikomasi bendrų, visiems dalyviams galiojančių <a href="/wiki/Vikipedija:Taisykl%C4%97s_ir_susitarimai" title="Vikipedija:Taisyklės ir susitarimai">taisyklių ir susitarimų</a>. Nauji dalyviai yra kviečiami apsilankyti <a href="/wiki/Pagalba:Turinys" title="Pagalba:Turinys">pagalbos puslapiuose</a> ir <a href="/wiki/Vikipedija:Bendruomen%C4%97" title="Vikipedija:Bendruomenė">bendruomenės portale</a>.
</p>

<div class="main-page-block-heading">Savaitės straipsnis</div>
<div class="main-page-block-contents"><figure class="mw-default-size" typeof="mw:File/Thumb"><a href="/wiki/Vaizdas:Franz_Xaver_Winterhalter_Family_of_Queen_Victoria.jpg" class="mw-file-description"><img src="//upload.wikimedia.org/wikipedia/commons/thumb/a/a1/Franz_Xaver_Winterhalter_Family_of_Queen_Victoria.jpg/220px-Franz_Xaver_Winterhalter_Family_of_Queen_Victoria.jpg" decoding="async" width="220" height="174" class="mw-file-element" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/a1/Franz_Xaver_Winterhalter_Family_of_Queen_Victoria.jpg/330px-Franz_Xaver_Winterhalter_Family_of_Queen_Victoria.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/a1/Franz_Xaver_Winterhalter_Family_of_Queen_Victoria.jpg/440px-Franz_Xaver_Winterhalter_Family_of_Queen_Victoria.jpg 2x" data-file-width="1890" data-file-height="1492" /></a><figcaption>Karlienė Viktorija su šeima (dail. <a href="/wiki/Franz_Xaver_Winterhalter" title="Franz Xaver Winterhalter">Franz Xaver Winterhalter</a>, 1846 m.)</figcaption></figure>
<p><b>Viktorijos epocha</b> (<a href="/wiki/Angl%C5%B3_kalba" title="Anglų kalba">angl.</a> <i>Victorian era</i>) – karalienės <a href="/wiki/Viktorija_(karalien%C4%97)" title="Viktorija (karalienė)">Viktorijos</a> valdymo laikotarpis <a href="/wiki/Jungtin%C4%97_Karalyst%C4%97" title="Jungtinė Karalystė">Jungtinės Karalystės</a> ir <a href="/wiki/Brit%C5%B3_imperija" title="Britų imperija">Britų imperijos</a> istorijoje nuo 1837 m. birželio 20 d. iki jos mirties 1901 m. sausio 22 d. Ši epocha prasidėjo po <a href="/wiki/Hanoveri%C5%B3_dinastija" title="Hanoverių dinastija">Hanoverių dinastijos</a> karalių <a href="/wiki/Jurgis_I" title="Jurgis I">Jurgio I</a>, <a href="/wiki/Jurgis_II" title="Jurgis II">II</a>, <a href="/wiki/Jurgis_III" title="Jurgis III">III</a> ir <a href="/wiki/Jurgis_IV" title="Jurgis IV">IV</a> valdymo epochos ir tęsėsi iki <a href="/wiki/Eduardas_VII" title="Eduardas VII">Eduardo</a> epochos, o jos antroji pusė sutapo su pirmąja žemyninės Europos <a href="/w/index.php?title=Gra%C5%BEioji_epocha&amp;action=edit&amp;redlink=1" class="new" title="Gražioji epocha (puslapis neegzistuoja)">Gražiosios epochos</a> (<a href="/wiki/Pranc%C5%ABz%C5%B3_kalba" title="Prancūzų kalba">pranc.</a> <i>Belle Époque</i>) dalimi. 
</p><p>Didžiojoje Britanijoje vyko įvairios <a href="/wiki/Liberalizmas" title="Liberalizmas">liberalizuojančios</a> politinės reformos, įskaitant <a href="/wiki/Rinkimai" title="Rinkimai">rinkimų</a> teisės išplėtimą. <a href="/w/index.php?title=Bulvi%C5%B3_badas&amp;action=edit&amp;redlink=1" class="new" title="Bulvių badas (puslapis neegzistuoja)">Bulvių (Didysis) badas</a> Viktorijos laikotarpio pradžioje sukėlė apie milijoną mirčių <a href="/wiki/Airija" title="Airija">Airijoje</a> (ši aplinkybė nedarė įtakos monarchės nuostatai <a href="/wiki/Airiai" title="Airiai">airių</a> tautos atžvilgiu). Britų imperija palaikė palyginti taikius santykius su kitomis <a href="/wiki/Did%C5%BEioji_valstyb%C4%97" title="Didžioji valstybė">didžiosiomis valstybėmis</a>. Ji dalyvavo įvairiuose kariniuose konfliktuose, daugiausia prieš mažesnes galias. Britų imperija šiuo laikotarpiu išsiplėtė ir tapo dominuojančia galia pasaulyje. Valdant Viktorijai užbaigtas <a href="/wiki/Indija" title="Indija">Indijos</a> prijungimas, įsitvirtinta <a href="/wiki/Egiptas" title="Egiptas">Egipte</a> ir <a href="/wiki/Piet%C5%B3_Afrika" title="Pietų Afrika">Pietų Afrikoje</a>, užvaldytos <a href="/wiki/Kinija" class="mw-redirect" title="Kinija">Kinijos</a> teritorijos, susikūrė persikėlėlių kolonijos <a href="/wiki/Australija" title="Australija">Australijoje</a> ir <a href="/wiki/Naujoji_Zelandija" title="Naujoji Zelandija">Naujojoje Zelandijoje</a>, dominijos teisės suteiktos <a href="/wiki/Kanada" title="Kanada">Kanadai</a>.
</p><p>Viktorijos laikų visuomenė vertino aukštą asmeninio <a href="/wiki/Elgesys" title="Elgesys">elgesio</a> standartą visuose visuomenės lygiuose. <a href="/wiki/Moral%C4%97" title="Moralė">Moralės</a> akcentavimas davė postūmį socialinei reformai, tačiau apribojo tam tikrų <a href="/wiki/Socialin%C4%97_klas%C4%97" title="Socialinė klasė">visuomenės grupių</a> laisvę. <a href="/wiki/Gerov%C4%97s_valstyb%C4%97" title="Gerovės valstybė">Gerovė</a> šiuo laikotarpiu išaugo, tačiau sekinantis <a href="/wiki/Neprievalgis" title="Neprievalgis">neprievalgis</a> išliko. Pirmą kartą Didžiojoje Britanijoje <a href="/wiki/Ra%C5%A1tingumas" title="Raštingumas">raštingumas</a> ir ugdymas vaikystėje tapo beveik visuotinis. Nors buvo bandoma pagerinti gyvenimo sąlygas, būstai lūšnynuose ir ligos išliko rimta problema.
</p><p>Šiuo laikotarpiu vyko reikšmingas mokslo ir technologijų vystymasis. Didžioji Britanija buvo pažengusi <a href="/wiki/Pramon%C4%97" title="Pramonė">pramonėje</a> ir ypač <a href="/wiki/In%C5%BEinerija" title="Inžinerija">inžinerijoje</a>, tačiau šiek tiek atsiliko <a href="/wiki/Menas" title="Menas">meno</a> ir <a href="/wiki/%C5%A0vietimas" title="Švietimas">švietimo</a> srityse. Didžiosios Britanijos gyventojų skaičius sparčiai augo, o Airijoje – ženkliai sumažėjo.

<p>Pirmajame ture kandidatas išrenkamas, jei surenka daugiau nei pusę balsų. Jei rinkimuose dalyvauja mažiau nei 40 proc. rinkėjų, kandidatas turi surinkti bent 20 proc. visų rinkėjų balsų. Jais negali būti renkami asmenys, nebaigę atlikti bausmės pagal teismo paskirtą nuosprendį, taip pat asmenys, teismo pripažinti neveiksniais. Be to, asmuo, kuris buvo pašalintas iš pareigų per apkaltą, gali būti išrinktas praėjus dešimt metų po apkaltos.
